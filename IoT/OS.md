# OPERATING SYSTEMS

### Raspberry Pi OS
* [Raspberry Pi OS](https://www.raspberrypi.org/software/operating-systems/) is the official operating system for the Raspberry Pi. It is based on Debian and is the most popular operating system for the Raspberry Pi. It is available in two versions: 32-bit and 64-bit. The 32-bit version is the most popular and is the version that is recommended for most users. The 64-bit version is only recommended for users who need to use 64-bit software. The 64-bit version is also only available for Raspberry Pi 3 and 4 models.
